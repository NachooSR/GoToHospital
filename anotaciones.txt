Gin

1) Que es Gin Context?
Es una estructura que Gin le pasa al Handler. Contiene toda la info que tiene que ver con una peticion HTTP

Cada campo permite una tarea diferente como por ejemplo

a) Leer datos del request (parámetros, query params, body, headers, etc.)

b) Escribir la respuesta (JSON, HTML, status code, headers, etc.)

c) Guardar información temporal para pasar entre middlewares

d) Abortar la ejecución si hay errores

e) Obtener el método (GET, POST, etc.) y la URL
